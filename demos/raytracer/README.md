# Mo Raytracer Demo

This demo renders a small PPM image with Mo code using `Float64` arithmetic,
structs, functions, loops, boolean operators, compound assignment, and standard
library file output.

Run it:

```sh
cargo run -- run demos/raytracer/main.mo
```

The program writes:

```text
/tmp/mo_raytracer.ppm
```

Open the image with any viewer that supports plain PPM files.
